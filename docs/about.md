# About

This site was built as a school assignment to explore [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) — a documentation framework on top of MkDocs.

## What is MkDocs Material?

Lorem ipsum dolor sit amet, consectetur adipiscing elit. MkDocs Material provides a polished, responsive theme with a huge set of built-in features:

- :material-palette: **Theming** — light/dark mode, custom colours and fonts
- :material-magnify: **Search** — instant full-text search with highlight
- :material-tab: **Navigation** — tabs, sections, back-to-top
- :material-code-tags: **Code blocks** — syntax highlighting, copy button, annotations
- :material-github: **GitHub integration** — repo link, edit button, deploy via Actions

## Tech stack

| Tool            | Purpose                        |
|-----------------|--------------------------------|
